> v3 기능 구현 완료 후 디자인 작업 시 참고.
> 프로토타입 HTML: `docs/design/home_v1.html`

---

## 설계 원칙

### 첫 화면이 답해야 할 질문

```
A. "오늘 출퇴근길에 뭐 들을까"   ← 메인 (더 자주 발생)
B. "이거 한 번 찾아볼까"          ← 보조
```

기존 v2까지는 B만 답하고 있었음. v3 홈은 **A를 메인으로**, B는 상시 접근 가능한 도구로 배치.

### 핵심 원칙

- **검색 → 큐레이션** 으로 패러다임 전환
- 시간대 인식 가능 (출퇴근, 점심, 저녁)
- 신규 사용자 / 미사용 사용자 / 활성 사용자별 변주
- 컴팩트한 상단바, 카드형 피드

## 섹션별 명세

### 1. 상단바 (Topbar)

```
[Paperis]               [⚙️] [👤]
  로고                  설정   프로필
```

- **로고만** + **아이콘 2개**. "안녕하세요" 인사말은 제거.
- sticky position, 스크롤 시에도 상단 고정
- 백드롭 블러 적용 (반투명 배경)

**아이콘별 진입점:**

| 아이콘 | 메뉴 |
|--------|------|
| ⚙️ 설정 | 내 저널 관리, API 키, 결제/구독 |
| 👤 프로필 | 로그아웃, 사용량 현황 (이번 달 X/3회) |

### 2. 검색바

- 상단 고정 영역 바로 아래
- 컴팩트하게 — 메인이 아닌 도구로 위치
- 탭하면 펼쳐져서 자연어 검색 가능 (iOS Spotlight 패턴)
- placeholder: `자연어로 논문 검색...`

### 3. 이어 듣기 카드 (조건부 노출)

**노출 조건:** 사용자에게 미완료 재생목록이 있을 때만.

```
┌──────────────────────────────────┐
│ ▶  ▶ 이어 듣기                   │
│    CIMT 용량 최적화 — 60시간...   │
│    ████░░░░░░░░░ 03:24 / 08:15   │
└──────────────────────────────────┘
```

- 카드 자체에 다른 카드와 구분되는 컬러 액센트
- 큰 재생 버튼 + 진행 바 + 남은 시간 표시
- 탭 시 즉시 재생 시작
- "출퇴근길 이어듣기" 시나리오의 핵심 카드

### 4. 내 저널 새 호 (가로 스크롤)

**카드 디자인:**

```
┌──────────┐
│  ●       │  ← 빨간 점 + glow: 아직 안 본 새 호
│   Arch   │
│   PM&R   │
│  ────    │
│  2026.04 │  ← 호 정보
│  24편    │  ← 논문 수 (+ OA 비율)
│  3편 OA  │
└──────────┘
```

- 가로 스크롤 carousel, scroll-snap
- 신규 호: `●` 빨간 점 + 미세 glow
- 탭 → 해당 호 탐색 화면으로 진입
- 우측 끝에 "전체 보기" 링크

### 5. 이번 분기 트렌드 (피처드)

- 자동 생성된 트렌드 브리핑 1개를 피처드로 노출
- 사용자의 가장 우선순위 높은 저널 기준
- 시즌 자동 계산 (현재 분기 또는 최근 완료된 분기)

```
┌──────────────────────────────────┐
│ Arch PM&R · 2026 Q1              │
│                                  │
│ 상지재활의 용량–반응 관계가      │
│ 다시 쓰이고 있다.                │
│                                  │
│ ▶ 편집장 브리핑 · 약 8분          │
└──────────────────────────────────┘
```

- headline 한 줄 또는 두 줄
- 청취 시간 명시 (task-match 행동에 대응)
- 우측 상단에 "다른 저널 →" 링크

### 6. 이어 읽기

- 최근 본 논문 가로 스크롤 또는 세로 리스트
- 각 카드: 저널명·호 + 논문 제목 (2줄까지)
- 탭 → 논문 상세로 점프
- DB에 reading history 필요 (4순위 구현)

### 7. 빠른 진입 (Quick Access)

```
┌──────┐ ┌──────┐ ┌──────┐
│  📅  │ │  🏷️  │ │  📈  │
│ 호   │ │ 주제 │ │트렌드│
│ 탐색 │ │      │ │ 탐색 │
└──────┘ └──────┘ └──────┘
```

- 3등분 그리드
- 위 카드들이 다 비어 있는 신규 사용자, 의도적 탐색하려는 사용자용
- 큐레이션 풀뷰로 진입

## 컨텍스트별 변주

### 신규 사용자 (내 저널 없음)

```
┌─────────────────────────────────┐
│  📚 관심 저널을 등록해보세요     │
│  새 호가 나오면 바로 알려드려요  │
│  [저널 등록하기 →]              │
└─────────────────────────────────┘
```

- "내 저널 새 호" 카드 자체가 안 보이고, 안내 카드로 대체
- 검색바를 좀 더 강조

### 미사용 7일 이상 사용자

```
"그동안 새 호가 12편 도착했어요"
[놓친 것 둘러보기 →]
```

### 시간대 인식 (옵션, 임팩트 큼)

| 시간대 | 강조 콘텐츠 |
|--------|-----------|
| 오전 7-9시 | "출퇴근길에 듣기 좋은 트랙" |
| 점심 12-13시 | "10분 안에 보기 좋은 abstract" |
| 저녁 18-20시 | "퇴근길 브리핑" |

## 시각 디자인 톤

### 방향

**에디토리얼 의학 저널 톤** — Apple News + FT Weekend.
의학 콘텐츠라 화려한 것보다 차분하고 격조 있는 분위기.

### 폰트

- **헤드라인·저널명**: Fraunces (세리프) — 학술지 느낌
- **본문·UI**: Pretendard — 한글 가독성 우수
- **숫자**: tabular-nums 적용 (호 날짜, 시간 표시)

### 색상

```
--bg:        #0a0a0a       /* 배경 */
--surface:   #141414       /* 카드 */
--surface-2: #1a1a1a       /* 호버 */
--border:    #222          /* 구분선 */
--text:      #f5f5f5       /* 본문 */
--text-2:    #999          /* 보조 */
--text-3:    #666          /* 보조2 */
--accent:    #ff5b3a       /* 액센트 (따뜻한 주황) */
--new:       #ff3b3b       /* 새 알림 */
```

차가운 푸른색보다 따뜻한 색을 선택. 의학 저널 + 차분함 + 따뜻함 톤.

### 인터랙션

- 모든 카드 호버 시 미세한 lift (transform: translateY(-1~2px))
- 페이지 진입 시 stagger 애니메이션 (위에서 아래로 순차 등장)
- 새 호 점은 미세한 glow

## 구현 우선순위

```
1순위  새 호 carousel + 이어 듣기 + 빠른 진입 (3개 카드)
       → 이것만으로도 기존 검색 화면보다 훨씬 가치 있음
2순위  이번 분기 트렌드 피처드
3순위  이어 읽기 (DB에 reading history 필요)
4순위  시간대/사용 패턴 인식
```

1순위 3개만 해도 첫 화면이 완전히 달라짐.
